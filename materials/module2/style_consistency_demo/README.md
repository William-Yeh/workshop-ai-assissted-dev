# style_consistency_demo

本目錄用來支撐 Module 2 的 `2.3.3 風格一致性` 教學。

## 目前包含
- `bad_patch_example.diff`
- `better_patch_example.diff`
- `neighbor_context.md`

3. 再看較好的收斂版本
4. 最後收成 first-layer review checklist
