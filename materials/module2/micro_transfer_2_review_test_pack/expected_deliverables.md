# Expected Deliverables

學員至少交出以下其中一項：

## A. Focused review notes
- 哪裡是高風險點？
- 哪裡需要先問清楚？
- 哪裡現在先不要改太大？

## B. Test sketch
- 這個 sketch 要區分哪兩條候選原因？
- 它要固定哪個關鍵預期？
- 為什麼現在值得先補這個 test？

## C. Style consistency review
- 哪些地方與既有 repo 寫法不一致？
- 哪些地方像 invent 了第二套寫法？
- 若只能保留既有 pattern，應怎麼收斂？

## D. PR / issue summary
- symptom
- candidate causes
- still-missing information
- validation order
- next suggested move
