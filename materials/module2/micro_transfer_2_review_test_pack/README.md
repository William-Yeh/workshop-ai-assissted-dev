# Micro Transfer 2｜Review / Test 補強 Pack

## 目的
這個小包用來支撐 Module 2 的第二個微型遷移：

> **第一層 review / test 補強：讓 AI 先幫你看風險與缺口**

它不是完整 debug 任務。  
它的目的，是讓學員練習把分析結果轉成：

- focused review
- 最小 test sketch
- style consistency 檢查
- PR / issue summary

---

## 本包內容
- `task_brief.md`
- `expected_deliverables.md`
- `review_questions.md`
- `style_drift_spotting.md`
