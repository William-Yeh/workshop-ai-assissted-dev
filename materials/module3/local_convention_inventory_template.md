# Module 3｜Local Convention Inventory

## 用途
先觀察 repo / 文件集裡重複出現的 pattern，再決定哪些值得升格成 local convention card。

## 請至少觀察五類
1. 命名
2. 結構
3. 測試
4. 文件 / summary / handoff
5. 驗證順序 / reviewer language
