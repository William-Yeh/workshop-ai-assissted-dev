# Teaching Modules Map

## 同一個 repo 的定版原則
- Module 1、Module 2、Module 3 放在**同一個 git repo**
- 共用同一個 app、同一個 domain、同一套執行環境
- 不拆成兩套重複 app
- 用不同資料夾、不同 task pack、不同 tests、不同 git 節點來區分模組

## 模組入口
- `modules/module1/README.md`
- `modules/module2/README.md`
- `modules/module3/README.md`

## 共用程式層
- `app/`
- `commerce_platform/`

## 模組教材層
- `materials/module1/`
- `materials/module2/`
- `materials/module3/`

## 模組測試層
- `tests/module1/`
- `tests/module2/`
- `tests/module3/`
