# Module 2 Git Navigation

## Recommended flow
1. `git checkout main`
2. Read `materials/module2/module2_task_pack.md`
3. Reset to a teaching checkpoint with a tag if needed:
   - `git checkout m2-baseline`
   - `git checkout m2-main-case-start`
   - `git checkout m2-micro1-start`
   - `git checkout m2-micro2-start`
4. Move into an exercise branch when needed:
   - `git checkout m2-test-gap-work`
   - `git checkout m2-style-bad-patch`
   - `git checkout m2-style-better-patch`

## Notes
- Tags are for stable teaching checkpoints.
- Branches are for interactive work in VSCode / Cline.
- The repo is still Module 2 work-in-progress, not final curriculum lock.
